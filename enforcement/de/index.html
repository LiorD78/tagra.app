FAQ

Frequently asked questions for enforcement authorities

What types of tachographs does TAGRA Control support?

TAGRA Control evaluates records from digital first-generation tachographs (G1) and smart second-generation tachographs (G2V2). Supported file formats: .DDD, .C1B, .V1B and ESM.

What does the driver card analysis reveal?

The program evaluates driving times, breaks, rest periods, working time and card-free driving. Every violation is displayed with exact date, time and duration. For G2V2 cards, GPS coordinates from the tachograph record are also available.

How quickly does analysis run at a roadside inspection?

Loading a driver card via DOWNLOAD BOX II S typically takes 10–60 seconds depending on the amount of data on the card. The analysis itself and display of the violation report is immediate after data loading.

Can TAGRA Control be used offline without an internet connection?

Yes. TAGRA Control works fully offline. An internet connection is only required for installing software updates. The inspection activity and data analysis works without network connection.

What devices does TAGRA Control run on?

The software runs on Windows 7 SP1 and newer (Windows 8.1, 10, 11 — both 32 and 64-bit). Suitable for laptops and desktop computers. Minimum hardware requirements are low.

Is training available in English for international enforcement authorities?

Yes. Truck Data Technology provides training in English for international enforcement authorities and partner institutions. Contact us at sales@tagra.app to arrange conditions and schedule.

How is legislative currency ensured?

The software is continuously updated with every change to relevant EU and national legislation. Updates are available to all valid licence holders. The program itself notifies you when an update is available.
