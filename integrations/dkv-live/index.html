Setup guide

Step by step

Setting up the DKV LIVE® import in TAGRA takes about two minutes. You only do it once — from then on, TAGRA pulls data automatically.

01
Open Data Import

In TAGRA's main window, click Import dat (Data Import) in the left-hand menu. This is where you set up the bridge between DKV LIVE® and TAGRA.

02
Add a new import

In the left panel of the Data Import screen, click Add import. A configuration window opens.

03
Configure the connection

Fill in the configuration window:

  • Import name — any label you like (e.g. “DKV”)
  • Folder — enter / (a single forward slash)
  • Run interval — how often TAGRA should fetch data (recommended: 1d for daily)
  • SFTP server — sftp.styletronic.eu
  • Port — 22
  • Username and password — your DKV LIVE® credentials

Confirm with OK.

04
Run the first import

Click on your new connection (it highlights in blue) and press Run import. TAGRA pulls the latest tachograph and driver card files from the DKV server immediately. When it finishes, switch back to the main screen and start your evaluation.

Automatic or manual?

Set it once,
forget about it

The Run interval field decides how often TAGRA fetches data from DKV LIVE®:

  • 1d — once a day (recommended)
  • 12h — every 12 hours
  • 6h, 3h, 1h — more frequent
  • 0s — manual only (you click Run import yourself)

We recommend at least daily so your data stays fresh and the legal 28 and 90-day download windows are always covered.

Why it matters 28 and 90 days are legal deadlines

EU regulations require driver card data to be downloaded at least every 28 days and tachograph unit data at least every 90 days.

With DKV LIVE® + TAGRA running on a daily schedule, those deadlines are met automatically. You also have a clean, archived trail for any inspection.

Common questions

FAQ

Which TAGRA versions support DKV LIVE®?
All current TAGRA versions — TAGRA 1, 2, 4, 6 and MAX — support DKV LIVE® import from version 345 onwards. The version number is shown at the bottom of the TAGRA window. If you have an older build, update via the in-app updater — updates are included in your annual licence.
Where do I get my DKV LIVE® credentials?
From your DKV Mobility account manager or sales representative. TAGRA needs a valid SFTP username and password for the DKV LIVE® server — the same ones you would use to access the service directly. We do not handle DKV account provisioning.
How often should TAGRA fetch data from DKV LIVE®?
We recommend at least once a day (interval 1d). That keeps your data current and ensures you're well within the 28-day and 90-day legal download windows. You can set it to every few hours if you prefer near-real-time evaluation — both work fine.
What if the import fails or throws an error?
First, double-check the SFTP server (sftp.styletronic.eu), port (22) and your credentials. If those are correct and the import still fails, contact us — phone +420 274 777 390 or email helpdesk@tdt.cz. The error message in TAGRA's import list helps us diagnose the problem quickly.
Do I need DKV LIVE® for remote tachograph download?
No. DKV LIVE® is one of several ways to get tachograph data into TAGRA — it's a great option if you already use DKV's telematics service. If you don't, TAGRA also accepts standard .DDD, .C1B and .V1B files from any other tachograph download tool. Contact us if you'd like a recommendation.
Is TAGRA officially partnered with DKV?
TAGRA imports data from the DKV LIVE® SFTP service using your own DKV credentials. This is a technical interoperability feature, not a formal partnership or endorsement. DKV LIVE® is a trademark of DKV Mobility.
Is the DKV LIVE® import included in my TAGRA licence?
Yes. The Data Import feature (including DKV LIVE® connectivity) is part of every current TAGRA licence at no extra cost. You only pay your normal TAGRA licence fee and your normal DKV LIVE® subscription — nothing more.
